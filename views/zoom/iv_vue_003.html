Deivo 10/20/2020 VueZoom

# Props 传入对象的所有 property

  • 如果想要把一个对象的所有property都作为props传入，可以使用不带参数的v-bind(取代 v-bind:prop-name)
obj: {
  	id:1,
    title: 'deivo'
}

<can-button v-bind="post"></can-button>

等价于：

<can-button v-bind:id="post.id" v-bind:title="post.title"></can-button>
